# upcoming-projects-table

MLA Advisory's Upcoming Project Table generator. Drop a HubSpot pull of pipeline/upcoming projects into Claude and say "run the upcoming table" — get back the table as Word or PowerPoint, sorted soonest launch first, with pipeline stage + estimated launch in the STATUS/TIMING column and a TOTAL units row.

Same layout for for-sale and rental product. The market belongs in the title/header, not per row — filter the pull by market before running (especially for PowerPoint, which fits ~13 rows per slide).

> Here's the upcoming highrise pull for Coquitlam — PowerPoint, title "Upcoming Concrete Condominium – Coquitlam".
