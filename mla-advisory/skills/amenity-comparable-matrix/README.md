# amenity-comparable-matrix

MLA Advisory's Amenity Comparables Matrix generator — **Word only**. Drop a HubSpot pull with a Project Amenities column and say "run the amenity matrix" — Claude maps each free-text amenity onto the locked 21-row taxonomy (showing you the mapping first), then builds the checkmark matrix with blue-shaded cells, TOTAL % column, STRATA FEES row, and per-project totals.

More than 7 projects chunk into additional matrices, one per page.
