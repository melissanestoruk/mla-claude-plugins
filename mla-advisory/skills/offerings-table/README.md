# offerings-table

MLA Advisory's Offering Table generator. Drop a HubSpot pull with offering data (deposits, commissions, incentives, strata fees, parking/storage, appliances) into Claude and say "run the offerings table" — get back the transposed Competition Offering table (projects as columns) as Word or PowerPoint.

Two variants: for-sale (Competition Offering) and rental (Rental Offering), each with its own locked attribute rows. Big comp sets chunk automatically — 5 projects per Word table (one per page), 4 per PowerPoint slide.

LATEST SALES ACTIVITY comes back as "-" — HubSpot doesn't track it, so it's filled in manually, like the asterisk footnotes.
