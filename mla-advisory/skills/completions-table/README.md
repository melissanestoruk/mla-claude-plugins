# completions-table

MLA Advisory's Completions Table generator. Drop a HubSpot pull of projects completing in a market and say "run the completions table" — get back the table in Word, or in PowerPoint get **two charts**: the table slide plus a Sold/Unsold by completion-quarter bar chart, both rebuilt from the pull.

Rows sort soonest completion first; TYPE maps Concrete → CON and Wood Frame → WF; the Word table shows percent sold per project and in the TOTAL row.
