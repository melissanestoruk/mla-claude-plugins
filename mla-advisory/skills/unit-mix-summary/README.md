# unit-mix-summary

Generates a unit mix summary table Word document for multi-family real estate competitive analysis.

Takes pasted project data (project name, unit types, unit counts, size ranges in SF) for one or more projects and produces a formatted .docx table with UNIT TYPE, %-INVENTORY, MIN-SIZE (SF), MAX-SIZE (SF), AVERAGE-SIZE, and MEDIAN-SIZE aggregated across all projects.

## When to use

Use this skill whenever unit mix data from multiple projects is pasted and a summary comparison table, competitive set analysis, or unit mix breakdown is needed — even if the user doesn't say "unit mix summary" explicitly.

## Output

Word document (.docx) saved to the Desktop via PowerShell Word COM automation.
